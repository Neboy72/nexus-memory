Features

Built for Agents,
Designed for Humans

Everything you need for persistent, cross-agent memory - zero lock-in.

Hybrid Retrieval

BM25 full-text search + vector similarity + RRF re-ranking. Precision meets recall.

Drift Detection

Automatically detects contradictory memories. Your agent never acts on outdated facts.

Zero Trust Security

Three access levels - public, trusted, private. Provenance tracking on every memory.

MCP Native

Model Context Protocol out of the box. Works with Hermes, OpenClaw, Claude Code, Cursor, and more.

Lifecycle Management

Memories age through fresh → active → stale → archived. Automatic cleanup of noise.

6 Embedding Providers

Voyage, OpenAI, Google, Jina, Ollama, sentence-transformers - auto-detected, zero config.
